Static Properties

analytics.filters.FilterBuilder.Predicate :
Returns true to indicate that a filter should be applied to a hit. This is the means by which you match your filters to hits with particular qualities.

NOTE: A predicate should not mutate a hit. If you want to mutate a hit, do it in a filter.
