analytics.filters.EventLabelerBuilder
Builder class for a labeler filter. Filters built using this class will auto-generate a label based on the the value supplied with the hit.

If an existing label was specified, it will be replaced by the new one unless the appendToExistingLabel() feature is specified, in which case the new label will be appended to the end of the existing label like so: "Existing Label - 256-512".

Example: If 205 were the value for a hit, this function would generate a label of '128-256'.

NOTE: Negative or zero values just get mapped to "<= 0". Non-numeric values get left as-is.
